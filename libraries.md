# Arduino Libraries

Install these via **Arduino IDE → Tools → Manage Libraries**

| Library | Author | Version | Install Name |
|---------|--------|---------|--------------|
| LiquidCrystal I2C | Frank de Brabander | ≥ 1.1.2 | `LiquidCrystal I2C` |
| WiFiS3 | Arduino | Included with R4 board package | Comes with **Arduino UNO R4 Boards** |

## Board Package

1. Open Arduino IDE → **Tools → Board → Boards Manager**
2. Search for `Arduino UNO R4`
3. Install **Arduino UNO R4 Boards** by Arduino
4. Select **Tools → Board → Arduino UNO R4 WiFi**
